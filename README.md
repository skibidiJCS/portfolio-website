# Jiacai Song Online CV

This is a static personal website. You can open `index.html` directly, or run a local server from this folder.

## Add your photos

Put your pictures in the `assets` folder with these filenames:

- `chess-rapid-2025-1st.jpg`
- `team-quebec-ontario-2025-1st.jpg`
- `chess-rapid-2024-3rd.jpg`
- `dragon-boat.jpg`
- `expo-sciences.jpg`
- `cycc-2025.jpg`
- `naycc-2025.jpg`

The current `assets` folder already includes web-ready versions of the photos.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
